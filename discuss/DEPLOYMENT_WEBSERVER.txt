## Web Server Configuration

1. Create a Gunicorn service file at /etc/systemd/system/discuss.service:
   ```
   [Unit]
   Description=Discuss Gunicorn Daemon
   After=network.target
   
   [Service]
   User=discuss
   Group=www-data
   WorkingDirectory=/home/discuss/discuss
   ExecStart=/home/discuss/discuss/venv/bin/gunicorn \
             --access-logfile - \
             --workers 3 \
             --bind unix:/home/discuss/discuss/discuss.sock \
             discuss.wsgi:application
   
   [Install]
   WantedBy=multi-user.target
   ```

2. Start and enable the Gunicorn service:
   ```
   sudo systemctl start discuss
   sudo systemctl enable discuss
   ```

3. Create an Nginx configuration file at /etc/nginx/sites-available/discuss:
   ```
   server {
       listen 80;
       server_name yourdomain.com www.yourdomain.com;
   
       location = /favicon.ico { access_log off; log_not_found off; }
       
       location /static/ {
           root /home/discuss/discuss;
       }
       
       location /media/ {
           root /home/discuss/discuss;
       }
       
       location / {
           include proxy_params;
           proxy_pass http://unix:/home/discuss/discuss/discuss.sock;
       }
   }
   ```

4. Enable the Nginx site:
   ```
   sudo ln -s /etc/nginx/sites-available/discuss /etc/nginx/sites-enabled/
   sudo nginx -t
   sudo systemctl restart nginx
   ```

5. Set up SSL certificates using Certbot (Let's Encrypt).
